Cascade of CSS Example

Subheading 1

Inheritence allows us to write more global styles on parent elements and then "fine-tune" the styles for some of the child elements as needed. Such approach makes writing CSS very efficient.

Subheading 2

Inheritance is a mechanism that's separate from the cascade: inheritance applies to the DOM (Document Object Model) tree, while the cascade deals with the style sheet rules. However, CSS properties set on an element via the cascade can beinherited by that element's child elements.
